## Mongodb

- create mongodb cloud
> create .env file in /api
```
  $git clone https://github.com/tlhaeroglu/noteapp-restapi
  $cd api/
  
  your .env file:
  DATABASE= your database name
  DATABASE_USERNAME= username
  DATABASE_PASSWORD= password
```
> save

```
npm install
npm start
```

> And go `front-end/` 
click `index.html`
